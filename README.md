# 📈 NanoStock: Симуляция Рынка(биржи) с ИИ-агентами

**NanoStock** — это мультиагентная среда для симуляции фондового рынка. Проект позволяет моделировать взаимодействие классических «шумовых» трейдеров и продвинутых агентов на базе нейросетей (LSTM и Linear) в условиях искусственных рыночных циклов.

---

## 🚀 Быстрый старт

### 1. Как это чудо запустить?
Убедитесь, что у вас установлен **Python 3.10**. Рекомендуется использовать виртуальное окружение.
1) установить библиотеки через pip install -r requirements.txt

2) в Simulation.py в начале функции main(), можно настроить параметры симуляции

3) Запустить Simulation.py, подождать около 1 минуты (в зависимости от кол-во шагов симуляции) и увидеть результат
   
Для обучения на GPU AMD RX 6600 используется стек **ROCm 6.4** и операционная система Fedora Workstation 43 с измененными параметрами для видеокарты.
