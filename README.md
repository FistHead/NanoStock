# Разработка ядра экономической симуляции мультиагентной среды «Мипл»

Веб-симуляция рынка, где миплы (агенты) торгуют, выбирают тактику и общаются друг с другом.
Движок построен поверх неизменённого `Core.py`, а решения и реплики выдают модели
`mrplip_17M_3` (нейросеть-трансформер) и `AffectorSeller` (граф влияния).

## Запуск

```powershell
cd D:\GitProjects\NanoStock\WebSite
..\.venv\Scripts\python.exe app.py
```

Затем открыть `http://localhost:5000`.

## Структура

| Файл | Назначение |
|------|------------|
| `app.py` | Flask: страницы и JSON-API |
| `engine.py` | движок симуляции поверх `Core.py` (свечи, чат, события) |
| `brains.py` | обёртки моделей `MrplipBrain` и `AffectorBrain` |
| `miple_model.py` | архитектура трансформера для загрузки чекпоинта |
| `templates/` | страницы: о проекте, симуляции, окно симуляции |
| `static/` | стили (жёлтая тема, светлая/тёмная) и скрипты (свечной график) |

## Возможности

- главная «О проекте» с переходом к симуляциям;
- список симуляций с переименованием и удалением;
- создание акций (вручную и случайно) и миплов (колесо личности, выбор модели, случайный мипл);
- авто-режим: собрать симуляцию одной кнопкой;
- график со свечами / линией / областью;
- переключатель над чатом: переписка, решения, активы, события;
- миплы видят реплики друг друга и отвечают на них, иногда происходят события рынка.

Симуляции хранятся в памяти процесса (на время работы сервера).
