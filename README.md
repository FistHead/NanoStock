# 📈 NanoStock: Симуляция биржи с ИИ-агентами

**NanoStock** — это мультиагентная среда для симуляции фондового рынка. [cite_start]Проект позволяет моделировать взаимодействие классических «шумовых» трейдеров и продвинутых агентов на базе нейросетей (LSTM и Linear) в условиях искусственных рыночных циклов[cite: 1, 4, 20].

---

## 🚀 Быстрый старт

### 1. Как это чудо запустить?
[cite_start]Убедитесь, что у вас установлен **Python 3.10**. Рекомендуется использовать виртуальное окружение.
1) установить библиотеки через pip install -r requirements.txt

2) в Simulation.py в начале функции main(), можно настроить параметры симуляции

3) Запустить Simulation.py
   
[cite_start]Для обучения на GPU AMD используется стек **ROCm 6.4**.
